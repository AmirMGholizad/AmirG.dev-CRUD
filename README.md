# CRUD Web App

A web application for performing Create, Read, Update, and Delete (CRUD) operations using FastAPI and PostgreSQL. This application is designed to be scalable, reliable, and easy to deploy using Docker and Nginx.

## Features

- **FastAPI**: For building the web application.
- **PostgreSQL**: For database management.
- **Nginx**: For reverse proxy to handle incoming traffic efficiently.
- **Docker**: For containerization, simplifying the deployment process.

## Technologies Used

- **Backend**: FastAPI
- **Database**: PostgreSQL
- **Web Server**: Nginx
- **Containerization**: Docker
- **Cloud**: AWS EC2

## API Endpoints

- **GET** `/items`: Retrieve a list of items.
- **GET** `/items/{id}`: Retrieve a specific item by ID.
- **POST** `/items`: Create a new item.
- **PUT** `/items/{id}`: Update an existing item by ID.
- **DELETE** `/items/{id}`: Delete an item by ID.

‼️ Don't forget to check the api docs (`\docs`) !!
